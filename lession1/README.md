# Lession 1

## 基本内容

- 选一个令人舒适的编辑器
- 创建第一个python脚本
    - 在脚本开头写上 `#!/usr/local/bin/env python`
    - 更改脚本文件属性 `chmod a+x file.py`
    - 尝试执行脚本文件 `./file.py` 或者 `python file.py` (*ps: 这里有点类似shell脚本*)
- print 函数输出的多个字符串用**", "**(逗号和空格)连接
- raw_input 函数读取一个字符串
